# Galleria

A static gallery generator that creates HTML photo galleries from NormPic manifests.

## Overview

Galleria reads photo manifests generated by NormPic and produces static HTML galleries with:
- Thumbnail generation and optimization
- Paginated gallery pages
- Per-gallery styling
- Non-JavaScript compatibility

## Documentation

See [galleria/doc/README.md](doc/README.md) for complete documentation.

## Setup
```bash
# Install dependencies (from parent project root)
uv sync

# Generate gallery from config
uv run python -m galleria --config config/galleria/wedding.json
```

## Development
```bash
# Run tests
uv run pytest galleria/

# Run with custom config
uv run python -m galleria --config path/to/config.json

# Development mode with verbose output
uv run python -m galleria --config config/galleria/wedding.json --verbose
```

## Features

- **Manifest-driven** - Reads NormPic JSON manifests
- **Thumbnail processing** - Generates optimized WebP thumbnails
- **Pagination** - Splits large galleries into pages (60 photos/page)
- **Theme support** - Customizable HTML/CSS templates
- **Progressive enhancement** - Works without JavaScript

## Configuration
```json
{
  "manifest_path": "/path/to/normpic/manifest.json",
  "output_dir": "output/galleries/wedding",
  "thumbnail_size": 400,
  "photos_per_page": 60,
  "theme": "minimal"
}
```

## Architecture

Galleria follows functional module organization:
- `generator/` - Gallery generation orchestration
- `processor/` - Image processing (thumbnails)
- `renderer/` - HTML/CSS generation
- `reader/` - Manifest parsing
